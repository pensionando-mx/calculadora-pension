# Calculadora de Pensión Ley 73

Calculadora profesional para estimar la pensión bajo la Ley del Seguro Social 1973.

**Accede a la calculadora:** https://pensionando-mx.github.io/calculadora-pension/

Características:
- Cálculo de SBC con las últimas 250 semanas
- Proyección de pensión con asignaciones familiares
- Módulo Modalidad 40
- Reportes en PDF
